# Thinkies: Cognitive Reasoning Skills for Claude Code

A comprehensive collection of thinking tools and reasoning techniques designed to help you approach problems systematically, think creatively, and make better decisions.

## Overview

Thinkies provides 46 specialized cognitive skills organized into 8 categories. Each skill offers structured approaches to different aspects of thinking, from creative exploration to rigorous analysis.

## Quick Start

Simply describe your problem or question, and Claude will automatically use relevant thinkies skills to help you think through it more effectively.

Examples:
- "I'm stuck on this architecture decision" ‚Üí May trigger `branch`, `weigh-options`, or `six-hats`
- "Something feels off about this solution" ‚Üí May trigger `follow-intuition` or `socratic`
- "Help me understand this complex system" ‚Üí May trigger `decompose` or `systems-thinking`
- "I need creative solutions" ‚Üí May trigger `scamper`, `lateral-think`, or `domain-transfer`

## How Skills Work

Each skill provides:
- **Clear instructions** - Step-by-step guidance
- **Concrete examples** - Real-world applications
- **When to use** - Situational triggers
- **Practical focus** - Immediate applicability

Skills can be used individually or combined into thinking workflows. Claude will automatically select and apply relevant skills based on your needs.

## Tips for Best Results

1. **Be specific about your challenge** - The more context you provide, the better Claude can select appropriate thinking tools.

2. **Embrace uncertainty** - Many skills help explore what you don't know yet. It's okay to start with vague feelings or hunches.

3. **Combine skills naturally** - Skills work well together. You might start with `wonder`, move to `branch`, then use `weigh-options`.

4. **Think alongside** - These skills are designed for collaborative thinking. Engage with the process rather than waiting for answers.

5. **Match skill to need** - Use exploration skills when stuck, analysis skills when evaluating, creativity skills when you need fresh ideas, etc.
